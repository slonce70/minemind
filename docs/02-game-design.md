# Game Design

## Core Loop
1. Гравець заходить як гість.
2. Обирає нік, аватар, мову.
3. Стартує соло-раунд або private room.
4. Відповідає на `8-12` запитань з таймером.
5. Отримує score, streak і подіум.
6. Натискає replay або йде в інший режим.

## Match Shape
- один раунд: `8` питань
- таймер на питання:
  - `easy` -> `20` секунд
  - `medium` -> `18` секунд
  - `hard` -> `15` секунд
- базові очки за правильну відповідь: `100`
- speed bonus: `timeLeft * 6`
- difficulty multiplier:
  - `easy` -> `1.0`
  - `medium` -> `1.25`
  - `hard` -> `1.5`
- пропуск або тайм-аут = `0` очок

## Content Style
- не суха енциклопедія
- формулювання прості, короткі, без надто вузьких trivia-detail
- пояснення після відповіді мають давати маленький факт, а не просто казати "це правильно"

## Room Rules
- усі гравці в кімнаті отримують однаковий набір питань
- host створює room code
- lobby має ready state
- фінальний подіум показує ranks `1/2/3`
- difficulty має бути видимою ще до старту матчу
