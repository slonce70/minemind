# Privacy Policy Draft

## Product
`MineMind` is a quiz game for children and families with a Minecraft-first launch category.

## Data We Use
- anonymous authentication identifier
- player nickname
- selected avatar preset
- chosen language
- quiz progress and match results
- room participation data for private rooms

## Data We Do Not Intentionally Collect In V1
- email address
- phone number
- profile photos
- chat messages
- precise location
- contact list
- payment data

## Why Data Is Used
- to restore the player profile
- to run solo and private room quiz sessions
- to calculate scores, streaks, and leaderboards
- to keep rooms synchronized between players

## Child Safety Notes
- V1 has no chat and no public user search
- avatars are chosen from a preset list
- nicknames are filtered for unsafe language
- public matchmaking is not part of the launch scope

## Retention Direction
- keep only the minimum data required to provide the quiz experience
- revisit retention windows before public launch

## Pre-Launch Tasks
- replace this draft with a hosted policy URL
- confirm data retention and deletion process
- align Apple and Google disclosures with the final backend configuration
