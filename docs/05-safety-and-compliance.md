# Safety And Compliance

## Audience
- стартова аудиторія: `8-12`

## Safe Defaults
- тільки `guest + nickname`
- preset avatars
- без чату
- без user search
- без profile photos
- без external ads
- без purchases у v1

## Nickname Rules
- довжина `2-16`
- дозволені тільки літери, цифри, пробіл, `_`, `-`
- локальний profanity blocklist
- з часом перенести filter у backend для server-side control

## Store / Policy Notes
- App Store і Google Play metadata мають прямо пояснювати, що social-функції обмежені та без чату
- privacy policy потрібна до launch even for guest mode
- branding не має виглядати офіційним Minecraft product
