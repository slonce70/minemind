## Long Description Draft
MineMind is a fast multiplayer quiz game built for kids and families who love Minecraft-style knowledge battles. Jump in as a guest, choose a safe nickname, and play short rounds about mobs, blocks, recipes, biomes, and survival basics.

Start with solo mode, then create a private room and invite friends with a simple code. Everyone gets the same questions, the same timer, and the same final podium.

MineMind is designed with child-friendly boundaries in mind: no chat, no ads in v1, no profile photos, and no public social graph.
